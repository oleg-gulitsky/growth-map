# Growth Map - Expo Snack Version

Упрощенная версия приложения "Карта роста" для Expo Snack.

## Что изменено для Snack

- Объединены все компоненты в один файл `App.js`
- Удалены зависимости, которые не работают в веб-версии:
  - `react-native-reanimated` (анимации)
  - `react-native-gesture-handler`
  - `expo-router` (заменен на простое состояние для модального окна)
- Упрощены package.json и app.json для совместимости со Snack
- Данные уроков встроены прямо в код

## Функциональность

- ✅ Список уроков с разными статусами (завершено, активно, заблокировано)
- ✅ Модальное окно при нажатии на заблокированный урок
- ✅ Визуальные индикаторы статусов
- ✅ Адаптивный дизайн

## Что не включено

- Анимации (требуют react-native-reanimated)
- Навигация через expo-router (заменена на простое модальное окно)
- Разделение на отдельные файлы (все в одном App.js для простоты)
